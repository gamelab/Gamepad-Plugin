Thumbstick Class

Defined in: plugins/gamepad-plugin-1.0.0.js:338
Module: Gamepad
Parent Module: Kiwi.Plugins

The Thumbstick class is used to keep track of each thumbstick. Currently it only tracks index and value. In future we are hoping to add functionality such as duration and more

Defined in plugins/gamepad-plugin-1.0.0.js:338

Parameters:

  • index Number

    This is the index of the axis mapped out on the controller.

  • name String

    The name that we give the button. We have based these off the Xbox 360 controller

  • [threshold] Number optional

    Between 0 - 1. This sets the threshold that a button must exceed before it is considered hit. Only works for triggers.
